# TechStore

## Descripción

TechStore es una página web sobre una tienda de tecnología desarrollada con HTML5 y CSS3.

El sitio permite consultar categorías de productos, productos destacados, una galería de imágenes, información del equipo y un formulario de contacto.

## Páginas del proyecto

- Inicio (`index.html`)
- Categorías (`categorias.html`)
- Productos (`productos.html`)
- Galería (`galeria.html`)
- Contacto (`contacto.html`)
- Formulario enviado (`enviado.html`)
- Nosotros (`nosotros.html`)
- Currículum (`juan-daniel.html`)

## Tecnologías utilizadas

- HTML5
- CSS3

## Características

- Menú de navegación común en todas las páginas.
- Galería de imágenes con Flexbox.
- Formulario de contacto con validación HTML.
- Tablas con `rowspan` y `colspan`.
- Uso de CSS externo, interno e inline.
- Diseño organizado mediante etiquetas semánticas HTML5.
